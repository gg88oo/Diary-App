# 📔 My Diary App

A beautiful desktop diary application with a calendar interface. Write daily entries, navigate between months, and save your thoughts - all in a clean, premium-looking app.

## ✨ Features

- Interactive calendar to select dates
- Write and save daily diary entries
- Navigate between months and years
- Automatic file organization (year/month folders)
- Premium, modern UI design
- Persistent storage - your entries never disappear

## 📋 Requirements

- Python 3.6 or higher
- PyQt6

## 🚀 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/diary-app.git
cd diary-app

# Install PyQt6
pip install PyQt6

# Run the app
python diary.py
